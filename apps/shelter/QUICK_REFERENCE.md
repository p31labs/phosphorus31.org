## 🔄 Common Workflows

### Processing a High-Voltage Message

1. **Paste message** into Shield tab
2. **Review BLUF** (Bottom Line Up Front) summary
3. **Check voltage** (color-coded gauge)
4. **Read translation** (if needed)
5. **Press to Reveal** raw text (3-second delay)
6. **If overwhelmed:** Click "Need Support?" → Select validation category

**Time:** ~30 seconds to 2 minutes

---

### Daily Check-In

1. Navigate to **Heartbeat** tab
2. Click **"Start Check-In"** (if banner appears)
3. Answer **15 questions** (sliders)
4. Review **status percentage** (0-100%)
5. Submit → Status attached to heartbeat

**Time:** ~2-3 minutes

**Thresholds:**
- **80-100%**: High resonance (Green)
- **60-79%**: Moderate (Yellow)
- **40-59%**: Low (Orange)
- **0-39%**: Critical (Red)

---

### Creating a Module

1. Navigate to **Module Maker** tab
2. Enter **intent** (natural language)
3. Click **"Generate Module"**
4. Review **Harmonic Linter Report**:
   - Resonance Score (target: 0.35 ± 0.05)
   - Spoon Cost (max: 12)
   - Violations (if any)
5. Click **"Press to Reveal"** to view/edit code
6. Edit in **Monaco Editor** (optional)
7. Click **"Deploy Module"** to install

**Time:** ~5-10 minutes

---

### Managing Modules

1. Navigate to **Modules** tab
2. View **enabled/disabled** modules
3. **Toggle** enable/disable as needed
4. Click **"View"** to see details
5. **Uninstall** if no longer needed
6. **Abdicate** to lock version (irreversible)

**Time:** ~1-2 minutes

---

### Connecting to the Mesh

1. Navigate to **Heartbeat** tab
2. **Copy** your connection code
3. **Share** with trusted peer (secure channel)
4. Peer **adds you** using your code
5. Connection **establishes automatically**
6. Monitor **peer status** in Heartbeat tab

**Time:** ~2 minutes

---

### First Light Verification

1. Navigate to **First Light** tab
2. Click **"Test"** on each payload
3. Verify **6 checks**:
   - ✅ BLUF Isolation
   - ✅ Genre Error Detection
   - ✅ Visual Pre-Cognition
   - ✅ Deep Processing Queue
   - ✅ Tetrahedron Symmetry
   - ✅ Somatic Grounding
4. **Node Broadcast** appears when all pass
5. **Share** broadcast with mesh

**Time:** ~5 minutes

---

### Somatic Grounding (4-4-8 Breathing)

1. Navigate to **Safe** tab
2. Click **"You Are Safe"** menu
3. Select **"Calming Breath (4-4-8)"**
4. Follow **haptic pulse**:
   - Inhale: 4 seconds
   - Hold: 7 seconds
   - Exhale: 8 seconds
5. Repeat **3-5 cycles**

**Time:** ~2-3 minutes

## 🎯 Quick Fixes

### Message Not Processing
1. Check **Shield** tab for errors
2. Verify **LLM provider** in Settings
3. Check **browser console** for errors
4. Try **refreshing** the page

### Module Won't Enable
1. Check **Harmonic Linter Report** for violations
2. Verify **spoon cost** < 12
3. Check **topology** is "delta" (not "wye")
4. Review **violations** in Module Manager

### Mesh Connection Failed
1. Check **internet connection**
2. Verify **PeerJS signaling server** is accessible
3. Check **firewall** settings
4. Try **regenerating** connection code

### Status Percentage Not Updating
1. Complete **Daily Check-In** in Heartbeat tab
2. Verify **all 15 questions** answered
3. Check **browser storage** (IndexedDB)
4. Try **clearing cache** and retrying

### High Voltage Message Overwhelming
1. Click **"Need Support?"** button
2. Select **validation category**:
   - 🧠 Amygdala Hijack / RSD
   - ⚡ Sensory Overload
   - 📡 Communication Friction
   - 🔋 Burnout / Metabolic Depletion
3. Follow **somatic directives**
4. Use **4-4-8 breathing**
