P31 Labs Website — Optimization Test

Use this page to verify all optimizations are working correctly.

1. Performance Tests

  • Open Chrome DevTools → Lighthouse → Run audit
  • Check Performance score (target: 90+)
  • Verify First Contentful Paint < 1.5s
  • Verify Largest Contentful Paint < 2.5s
  • Verify Cumulative Layout Shift < 0.1
  • Check Network tab for font loading (should use font-display: swap)
  • Verify DNS prefetch/preconnect in Network tab

2. SEO Tests

  • View page source → Search for application/ld+json
  • Verify structured data is valid (use Google Rich Results Test)
  • Check meta tags in <head> (Open Graph, Twitter)
  • Verify all products have microdata (itemscope, itemprop)
  • Test with Schema.org Validator

3. Accessibility Tests

  • Press Tab key → Skip link should appear at top
  • Navigate with keyboard only (Tab, Enter, Arrow keys)
  • Check focus indicators (green outline on focus)
  • Run WAVE accessibility audit
  • Run axe DevTools audit
  • Test with screen reader (NVDA, JAWS, VoiceOver)
  • Verify heading hierarchy (h1 → h2 → h3)
  • Check color contrast (WCAG AA minimum)

4. CSS Performance Tests

  • Open DevTools → Performance tab → Record
  • Scroll page and check for layout shifts
  • Hover over cards/buttons → Check for smooth transitions
  • Verify contain property in Computed styles
  • Check will-change on animated elements
  • Test with reduced motion preference (Settings → Accessibility)

5. Mobile Tests

  • Test on real mobile device
  • Use Chrome DevTools device emulation
  • Verify responsive breakpoints (768px, 480px)
  • Check touch targets (minimum 44x44px)
  • Test viewport meta tag (no horizontal scroll)
  • Verify theme color shows in mobile browser

6. Browser Compatibility

  • Chrome/Edge (latest)
  • Firefox (latest)
  • Safari (latest)
  • Mobile Safari (iOS)
  • Chrome Mobile (Android)
  • Check CSS Grid support
  • Check CSS Custom Properties support

7. Network Tests

  • Throttle to 3G in DevTools → Test loading
  • Check DNS prefetch timing in Network tab
  • Verify font loading doesn't block rendering
  • Test with offline mode (Service Worker if added)
  • Check HTTP/2 server push (if configured)

8. Content Verification

  • All brand colors correct (#2ecc71, #60a5fa, #050510)
  • All links work (internal anchors, external links)
  • All images have alt text (when added)
  • Contact information correct (will@p31ca.org)
  • No OPSEC violations (no surnames, addresses)
  • Brand voice consistent throughout

Quick Test Commands

Run these in browser console:

// Check for structured data
JSON.parse(document.querySelector('script[type="application/ld+json"]').textContent)

// Check CSS containment
getComputedStyle(document.querySelector('.product-card')).contain

// Check will-change
getComputedStyle(document.querySelector('.btn')).willChange

// Check font-display
fetch('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap')
  .then(r => r.text())
  .then(t => console.log(t.includes('font-display:swap')))

// Check skip link
document.querySelector('.skip-link')
