Accessibility

P31 Labs is designed for everyone. We strive to meet or exceed WCAG 2.1 AA standards.

WCAG 2.1 AA

Compliance Statement

P31 Labs aims to be fully compliant with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. We regularly audit our website, documentation, and applications to ensure they are perceivable, operable, understandable, and robust for all users.

Our hardware (Node One) includes haptic and audio feedback, and our software supports screen readers, keyboard navigation, and high contrast modes.

This page itself is built with accessibility as a primary requirement – including a skip link, proper heading hierarchy, and ARIA labels.

Accessibility Features

⌨️

Keyboard Navigation

All interactive elements are focusable and operable with keyboard (Tab, Enter, Space, Arrow keys).

🔊

Screen Reader Compatible

Semantic HTML, ARIA labels, and alt text ensure compatibility with NVDA, JAWS, and VoiceOver.

👁️

High Contrast Mode

Our color scheme meets WCAG contrast requirements. We also support system high contrast modes.

🎬

Reduced Motion

We respect the `prefers-reduced-motion` media query to disable non‑essential animations.

📱

Responsive Design

Pages adapt to different screen sizes and zoom levels without loss of content or functionality.

📄

Clear Structure

Proper heading hierarchy, landmarks, and skip links help with navigation.

Keyboard Shortcuts

The following keyboard shortcuts are available throughout the site:

Skip to content Tab
Main navigation Tab / Arrow keys
Activate button/link Enter / Space
Close dialogs Escape
Search (where available) / (slash)

Known Issues & Roadmap

We are continuously improving. Here are current known issues and our plans to address them:

  • Wallet graphs – Some charts may not be fully accessible to screen readers. We plan to replace them with textual summaries by Q2 2026.
  • Games page – Some embedded game demos may not yet support switch control. We are adding ARIA and keyboard handlers in the next release.
  • PDF documents – Some downloadable PDFs are not fully tagged for screen readers. We will provide HTML versions alongside them.

Our roadmap includes specific accessibility milestones. If you encounter an issue not listed, please report it.

Reporting Accessibility Issues

We welcome feedback. If you encounter any barrier to using our website, hardware, or software, please let us know:

  • Email: a11y@p31labs.org
  • GitHub: Open an issue on our website repository
  • Community chat: Join our Discord and use the #accessibility channel

We aim to respond to accessibility reports within 3 business days.

Screen Reader Compatibility

We regularly test with:

  • NVDA (Windows, Firefox/Chrome)
  • JAWS (Windows, Edge)
  • VoiceOver (macOS, Safari)
  • Orca (Linux)

We recommend using the latest versions of these screen readers for the best experience.

Display Preferences

High Contrast

The site automatically adapts to your system's high contrast setting. All text remains readable against its background. If you notice any issues, please report them.

Reduced Motion

If you have enabled "prefers-reduced-motion" in your operating system, all animations and transitions are disabled or minimized. For example, the pulsing dot on the roadmap page will not pulse.

Still have questions?

Our accessibility coordinator can be reached at a11y@p31labs.org.

We are committed to making P31 Labs accessible to everyone. Thank you for helping us improve.
