03 — Why This Exists

The Problem

"We do not have assistive technology for the largest disability demographic on the planet. We have wellness apps."
15–20% of the global population is neurodivergent
55% of commercial AT wearables for autism have undefined FDA status (PMC 2024)
0 open-source cognitive prosthetics existed before this project

Wheelchair ramps exist because someone in a wheelchair demanded them. Screen readers exist because a blind engineer built one. Closed captions exist because Deaf advocates sued for them.

There is no equivalent infrastructure for cognitive disability. No communication prosthetic that processes emotional subtext. No executive function scaffold that works offline. No sensory regulation tool that doesn't require a smartphone and a subscription.

P31 builds them. Open source. Apache 2.0. Mesh-resilient. Because the person who needs it most is often the person least able to navigate the system that would provide it.
