## 🏥 FDA Regulatory (Parallel Track)

### Registration
- [ ] Register establishment with FDA ($11,423/year - may qualify for hardship waiver)
- [ ] List device with FDA (free)
- [ ] Apply for financial hardship waiver if ≤$1M gross receipts

### Quality Management System
- [ ] Implement 21 CFR Part 820 Quality System Regulation
- [ ] Document Design Controls (§ 820.30)
- [ ] Create Design History File
- [ ] Document ESP32 as SOUP (Software of Unknown Provenance) per IEC 62304

### Compliance
- [ ] Comply with labeling requirements (21 CFR Part 801)
- [ ] Set up Medical Device Reporting (21 CFR Part 803)
- [ ] Plan EMC testing (IEC 60601-1-2)
- [ ] Conduct cybersecurity risk assessment (FDA 2023 guidance)

### Partnerships
