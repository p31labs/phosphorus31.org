# P31 User Guide

**For everyone - from 6 to 70+**

## Quick Start

### First Time Setup

1. **Open The Scope** - Your dashboard
2. **Click "Simple"** - For easy-to-use interface
3. **Click "♿ Settings"** - To customize for you

### For Children (6+)

1. Click **"Simple"** button
2. Click **"♿ Settings"**
3. Click **"👶 Child Mode"**
4. You're ready! Large buttons, clear text, sound feedback

### For Seniors (70+)

1. Click **"Simple"** button
2. Click **"♿ Settings"**
3. Click **"👵 Senior Mode"**
4. You're ready! Extra-large text, high contrast, reduced motion

### For Everyone Else

1. Click **"Simple"** button
2. Click **"♿ Settings"**
3. Customize:
   - Text size (Small to Extra Large)
   - Contrast (Normal or High)
   - Sound feedback (On/Off)
   - Vibration feedback (On/Off)
   - Voice commands (On/Off)

## Using Simple Buffer

### Sending a Message

1. Click **"Simple"** button
2. Type your message in the big box
3. Click **"Send Message"** button
4. You'll hear a sound when it's sent!

### Using Voice Commands

1. Make sure Voice Commands are enabled in Settings
2. Click the **🎤 Speak** button
3. Say: **"Send message [your message]"**
4. The message will be sent automatically!

### Keyboard Shortcuts

- **Alt + 1** = Switch to Child Mode
- **Alt + 2** = Switch to Senior Mode
- **Alt + 3** = Switch to Standard Mode
- **Alt + S** = Toggle Simplified UI
- **Alt + C** = Toggle High Contrast
- **Alt + +** = Increase text size
- **Alt + -** = Decrease text size

## Features

### Text Sizes

- **Small** (14px) - For advanced users
- **Medium** (16px) - Default size
- **Large** (20px) - Recommended for children
- **Extra Large** (24px) - Recommended for seniors

### Contrast Modes

- **Normal** - Standard colors
- **High** - Maximum contrast (black/white/yellow)

### Feedback Options

- **Sound Feedback** - Hear when actions complete
- **Vibration Feedback** - Feel when actions complete
- **Screen Reader** - Spoken interface for vision impairments
- **Voice Commands** - Speak instead of type

## Tips

### For Children
- Use Simple Mode
- Enable sound feedback
- Use voice commands
- Keep it colorful and fun

### For Seniors
- Use Simple Mode
- Set text to Extra Large
- Enable high contrast
- Enable sound and vibration
- Reduce motion for comfort

### For Everyone
- Start with Simple Mode
- Adjust settings as needed
- Use keyboard shortcuts
- Test what works for you
