Code Example

Initializing the Node One mesh network:

import { NodeOne } from '@p31/node-one';

const node = new NodeOne({
  frequency: 915, // MHz
  meshId: 'p31-delta',
  haptics: true
});

node.on('message', (msg) => {
  console.log('Received:', msg);
});

await node.start();

Detailed Topics

Mesh Routing Protocol (Phenix)

The Phenix Navigator uses a modified AODV (Ad‑hoc On‑Demand Distance Vector) protocol with topology‑aware metric. Each node maintains a routing table with signal strength and path reliability. For full specification, see Routing Protocol.

Haptic Feedback Patterns

Node One supports 123 distinct haptic patterns defined in the DRV2605L library. Patterns are grouped into categories: alerts, confirmations, navigation, and data encoding. Refer to the Haptics Reference for waveform tables.
