Architecture

System Design

Three-layer stack. Each layer works independently. Together they form a mesh — no single point of failure.

Layer 1: Node One (Hardware)

ESP32-S3 microcontroller running ESP-IDF (FreeRTOS). LoRa mesh networking via Meshtastic. DRV2605L haptic engine. SSD1306 OLED display.

Communication: LoRa 915MHz ISM band. Range: 1–10km line-of-sight. Mesh protocol: Meshtastic. No internet required.
Haptics: "The Thick Click" pattern for sensory regulation. 8-bit waveform library. I2C interface to DRV2605L.
Power: 18650 Li-ion (2500mAh). USB-C charging. Deep sleep modes. Battery monitoring.

Layer 2: The Buffer (Software)

FastAPI backend with Redis cache. Local LLM (Ollama) for pattern detection. WebSocket for real-time updates.

API Endpoints: POST /api/v1/assess (voltage assessment), GET /api/v1/history (accommodation log), WebSocket /ws (real-time updates).
Pattern Detection: URGENCY, COERCION, SHAME, FALSE AUTHORITY, THREATS, EMOTIONAL LEVER. Voltage scale: 0–10.
Storage: SQLite for local-first operation. Redis for caching. Optional encrypted cloud sync.

Layer 3: The Scope (Interface)

React + Three.js frontend with Tauri desktop wrapper. Capacitor for mobile. Vite build system.

Components: Medication tracker, spoon economy, coherence monitoring, ping grid, 3D mesh visualization.
Data Sources: The Buffer (WebSocket), Node One (LoRa/USB), local storage (IndexedDB).
Accessibility: WCAG 2.1 AA. Keyboard navigation. Screen reader support. High contrast mode.
