## SECTION 2: DEVICE DESCRIPTION

**Device Name:** P31 Compass / Node One (P31 NodeZero)

**FDA Classification:** 21 CFR §890.3710 - Powered Communication System (Product Code ILQ, Class II, 510(k) Exempt)

**Technical Specifications:**

- Microcontroller: ESP32-S3-DevKitC-1
- Haptic Driver: DRV2605L (I2C address 0x5A)
- Connectivity: WiFi, BLE, LoRa (RFM95W 915MHz)
- Display: SSD1306 OLED 128x64
- Battery: 3.7V LiPo (2000mAh target, 7+ day life goal)
- Form Factor: Wrist-worn or pocket-portable

**Purpose Statement (approved claims only):**

"The device is a powered communication system and cognitive prosthetic device designed to support executive function for individuals with neurodivergent processing differences. It provides sensory regulation through programmable haptic feedback patterns, long-range mesh communication via LoRa for caregiver connectivity, and integration with digital assistive interfaces."

**NOT a medical treatment device. Assistive technology under ADA protections.**
