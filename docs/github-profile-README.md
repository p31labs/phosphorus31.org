### The Name

**P31** = Phosphorus-31, the only stable phosphorus isotope. NMR-active, quantum spin 1/2, the backbone of ATP.

**Ca9(PO4)6** = the Posner molecule — the hypothesized substrate for quantum cognition in biological systems.

[phosphorus31.org](https://phosphorus31.org) = the science.
