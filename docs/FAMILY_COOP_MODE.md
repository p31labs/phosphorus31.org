## Overview

Family Co-Op Mode is a special game mode designed for the family tetrahedron: Will, Co-parent, Bash, and Willow. It teaches collaboration, geometric principles, and family bonding through shared building challenges.

## How It Works

### 1. Family Setup

Each family member gets a role:
- **Foundation** (Will) — Builds the base, ensures stability
- **Structure** (Co-parent) — Builds the main structure, connects pieces
- **Connection** (Bash) — Connects parts, creates tetrahedron topology
- **Completion** (Willow) — Finishes structure, tests stability

### 2. Challenge Selection

Choose from family challenges:
- **The Family Foundation** (Easy) — Build with 4 vertices, 6 edges
- **The Family Bridge** (Medium) — Build a bridge connecting two points
- **The Stable Family** (Medium) — Build a structure that holds weight
- **The Family Tower** (Hard) — Build the tallest stable tower
- **The Family Dome** (Hard) — Build a geodesic dome together

### 3. Collaborative Building

- **Each person places pieces** — Everyone contributes
- **Work together** — Communicate, support each other
- **See progress** — Watch the structure grow
- **Test together** — See if it's stable

### 4. Story Mode

Each challenge has a story that teaches:
- **Geometric principles** — What is a tetrahedron?
- **Family concepts** — How families are like structures
- **Collaboration** — Why working together matters
- **Real-world application** — How this applies to life

## Family Challenges

### Challenge 1: The Family Foundation

**Difficulty:** Easy  
**Time:** 20 minutes  
**Story:** Learn how families are like tetrahedrons - four points, all connected.

**Objective:** Build a structure with exactly 4 vertices (one per family member) and 6 edges connecting them.

**Requirements:**
- 4 vertices (one per person)
- 6 edges (connections)
- 50+ stability
- Maxwell's Rule: Valid
- Use at least 2 different materials

**Rewards:**
- 50 LOVE tokens (shared)
- "Family Foundation" badge
- Individual role badges
- Bronze Family Trophy

---

### Challenge 2: The Family Bridge

**Difficulty:** Medium  
**Time:** 25 minutes  
**Story:** Learn how families connect us, even when we're apart.

**Objective:** Build a bridge connecting two points, with 60+ stability and at least 4 different colors.

**Requirements:**
- Bridge connects two points
- 60+ stability
- 4 different colors
- Each family member contributes
- Maxwell's Rule: Valid

**Rewards:**
- 60 LOVE tokens (shared)
- "Family Bridge" badge
- Rainbow materials unlock
- Bridge blueprint

---

### Challenge 3: The Stable Family

**Difficulty:** Medium  
**Time:** 30 minutes  
**Story:** Learn how families support each other, like structures support weight.

**Objective:** Build a structure with 70+ stability that can hold weight without collapsing.

**Requirements:**
- 70+ stability
- Passes physics test (holds weight)
- Maxwell's Rule: Valid
- Use at least 3 different materials
- Each person contributes to stability

**Rewards:**
- 75 LOVE tokens (shared)
- "Stable Family" badge
- Silver Family Trophy
- Advanced materials unlock

---

### Challenge 4: The Family Tower

**Difficulty:** Hard  
**Time:** 40 minutes  
**Story:** Learn that tall structures need strong foundations, just like families.

**Objective:** Build the tallest stable tower (80+ stability) with at least 20 pieces.

**Requirements:**
- At least 20 pieces
- 80+ stability
- Tallest possible
- Each person contributes equally
- Passes physics test

**Rewards:**
- 100 LOVE tokens (shared)
- "Family Tower" and "Sky Reachers" badges
- Gold Family Trophy
- Tower blueprint
- Special materials unlock

---

### Challenge 5: The Family Dome

**Difficulty:** Hard  
**Time:** 45 minutes  
**Story:** Learn how families protect us, like domes protect what's inside.

**Objective:** Build a geodesic dome with 90+ stability using tetrahedron and octahedron pieces.

**Requirements:**
- Geodesic dome shape
- 90+ stability
- Use tetrahedron and octahedron pieces
- Each person contributes significantly
- Passes all physics tests

**Rewards:**
- 150 LOVE tokens (shared)
- "Family Dome", "Dome Builders", "Protectors" badges
- Platinum Family Trophy

## Roles and Responsibilities

### Foundation Role (Will)
- **Build the base** — Create the foundation
- **Ensure stability** — Make sure it won't fall
- **Place first pieces** — Start the structure
- **Special power:** Can place foundation pieces faster, gets stability bonus

### Structure Role (Co-parent)
- **Build main structure** — Create the body
- **Connect pieces** — Link everything together
- **Ensure integrity** — Make sure it's strong
- **Special power:** Can see connection points, gets connection bonus

### Connection Role (Bash)
- **Connect parts** — Link different sections
- **Create topology** — Form the tetrahedron shape
- **Ensure connections** — Make sure all vertices connect
- **Special power:** Can see all connection opportunities, gets topology bonus

### Completion Role (Willow)
- **Finish structure** — Add final touches
- **Test stability** — See if it's strong
- **Add beauty** — Make it colorful and pretty
- **Special power:** Can test structure anytime, gets completion bonus

## Getting Started

### Step 1: Set Up Family

1. Create profiles for each family member
2. Assign roles (Foundation, Structure, Connection, Completion)
3. Choose favorite colors for each person

### Step 2: Start First Challenge

1. Select "The Family Foundation" (recommended first challenge)
2. Read the story together
3. Understand the objective
4. Review requirements

### Step 3: Build Together

1. **Foundation** (Will) — Start with the base
2. **Structure** (Co-parent) — Build the main structure
3. **Connection** (Bash) — Connect all the parts
4. **Completion** (Willow) — Finish and test

### Step 4: Complete and Celebrate

1. Test the structure
2. Check if requirements are met
3. Celebrate together!
4. Earn rewards (LOVE tokens, badges, trophies)
5. Unlock next challenge

## Tips for Success
