# Accessibility Keyboard Shortcuts

Quick reference for keyboard shortcuts in P31.

## Mode Switching

- **Alt + 1** - Switch to Child Mode
- **Alt + 2** - Switch to Senior Mode  
- **Alt + 3** - Switch to Standard Mode

## Interface Controls

- **Alt + S** - Toggle Simplified UI
- **Alt + C** - Toggle High Contrast
- **Alt + +** - Increase text size
- **Alt + -** - Decrease text size

## Usage Tips

- All shortcuts use the **Alt** key
- Shortcuts work from anywhere in The Scope
- Changes apply immediately
- Settings persist across sessions
