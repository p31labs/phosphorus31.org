# Kids Mode - Built for Bash and Willow
## Safe, Educational, Fun Game Engine Features

**"Built with love and light. As above, so below."** 💜

## Overview

Kids Mode is a special set of features designed specifically for children, with Bash (age 9, Node One) and Willow (age 6, Node Two) in mind. It provides:

- **Age-appropriate challenges** — Sprout tier for Willow, Sapling tier for Bash
- **Educational story mode** — "The Tetrahedron Quest" teaches geometric principles
- **Family co-op mode** — Build together as a family tetrahedron
- **Privacy-first** — All data stays local, no tracking
- **Safe settings** — Parent-controlled time limits and content filters
- **Break reminders** — Energy-aware gameplay

## Features

### 1. Educational Story Mode

**"The Tetrahedron Quest"** — A narrative that teaches geometric principles:

- **Chapter 1: The First Shape** (Sprout)
  - Learn what a tetrahedron is
  - Build your first structure
  - Unlock: Rainbow colors

- **Chapter 2: The Rainbow Bridge** (Sprout)
  - Learn about stability
  - Build a colorful bridge
  - Unlock: Family chapter

- **Chapter 3: The Family Tetrahedron** (Sapling)
  - Understand family structure (4 vertices)
  - Build with 4 connection points
  - Unlock: Maxwell's Rule

- **Chapter 4: Maxwell's Secret** (Sapling)
  - Learn E ≥ 3V - 6
  - Build strong structures
  - Unlock: Family co-op

- **Chapter 5: Building Together** (Family)
  - Work with family
  - Build something amazing together
  - Complete the quest!

### 2. Kid Profiles

Each child gets their own profile:
- **Display name only** — Privacy first, no real names stored
- **Age-appropriate tier** — Automatically assigned
- **Favorite colors** — Personalized palette
- **Achievements** — Badges and unlocks
- **Play time tracking** — Local only, for time limits

### 3. Safe Settings

All settings are parent-controlled:
- **Content filter** — Always "strict" for kids
- **Time limits** — Optional daily limits (default: 1 hour)
- **Break reminders** — Every 20 minutes
- **Parent approval** — Required for certain actions
- **Sharing** — Family-only, local only
- **Cloud sync** — Parent must enable

### 4. Family Co-Op Mode

Build together as a family tetrahedron:
- **4 players required** — Family structure (4 vertices)
- **Collaborative challenges** — Work together
- **Shared progress** — Family achievements
- **No strangers** — Only family members
- **Tetrahedron topology** — Learn family structure

**See also:** [Family Co-Op Mode](FAMILY_COOP_MODE.md) for detailed family challenges and roles

### 5. Privacy Protection

- **Local-only storage** — All data stays on device
- **No tracking** — Zero analytics, zero telemetry
- **Encrypted saves** — Structures encrypted at rest
- **No accounts** — Play offline, no sign-up
- **Parent-controlled** — All sharing requires approval
