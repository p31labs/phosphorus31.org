# The Centaur

Backend AI protocol system. Human + Synthetic = Centaur.

## Overview

The Centaur is not a pair, but a protocol. It represents the interface between human operators and synthetic intelligence. The protocol is vendor-agnostic - cloud when connected, local when sovereign, hybrid when in between.

## Architecture

The Centaur protocol consists of:
- **The +**: The interface (Node One, The Buffer, The Thick Click)
- **Human Operator**: The biological vertex
- **Synthetic Body**: The AI system (Claude, Gemini, local models, etc.)
- **Protocol Layer**: Abstraction over specific AI implementations

## Features

- Vendor-agnostic AI integration
- Cloud/Local/Hybrid operation
- Quantum brain integration
- Decision engine
- Knowledge graph (Neo4j)
- SOP generation
- Consciousness monitoring

## Setup

See [SUPER-CENTAUR/setup.md](../SUPER-CENTAUR/setup.md) for complete setup instructions.

### Prerequisites

- Node.js 18+
- Docker (for services)
- PostgreSQL or SQLite
- Neo4j (for knowledge graph)
- Redis (for caching)

### Quick Start

```bash
cd SUPER-CENTAUR
npm install
npm run dev
```
