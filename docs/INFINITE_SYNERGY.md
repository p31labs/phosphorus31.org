## Mathematical Properties

### Synergy Formula

```
S(n) = 1.0 × (1.618^n) × (0.95^n)
     = 1.0 × (1.5371^n)
```

### Total Synergy

```
Total = Σ S(i) for i = 0 to n
      = Σ 1.0 × (1.5371^i)
```

### Fractal Dimension

```
D = log(N(n+1) / N(n)) / log(4)
  ≈ 1.585
```

Where:
- `N(n)` = number of nodes at level n
- `4` = scaling factor (4 children per node)
