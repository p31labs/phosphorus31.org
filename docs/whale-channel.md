# Whale Channel

LoRa 915MHz mesh network for infrastructure-independent communication.

## Overview

Whale Channel is the LoRa mesh network layer operating at 915MHz. It provides infrastructure-independent communication for Node One devices, enabling mesh networking without cellular or WiFi dependency.

## Features

- LoRa 915MHz mesh networking
- Infrastructure independence
- Low power operation
- Long range communication
- Mesh topology support
- Encrypted communication

## Architecture

Whale Channel uses:
- **LoRa Protocol**: Long-range radio
- **915MHz Band**: ISM frequency
- **Mesh Topology**: Multi-hop routing
- **Encryption**: End-to-end encrypted

## Hardware

Whale Channel requires:
- LoRa transceiver (on Node One)
- Antenna (915MHz)
- Power management

## Integration

Whale Channel integrates with:
- **Node One**: Hardware device
- **The Buffer**: Message routing
- **Ping**: Connection monitoring
- **The Centaur**: Backend communication

## Configuration

Whale Channel configuration includes:
- Frequency: 915MHz
- Spreading factor
- Bandwidth
- Coding rate
- Transmission power
