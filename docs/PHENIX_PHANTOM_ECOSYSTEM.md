## Part III: The Cognitive Shield

### 3.1 The Threat: ACR and Digital Surveillance

Smart TVs use **Automatic Content Recognition (ACR)** to capture pixel-level fingerprints of displayed content. This data is used for:

- Behavioral profiling
- Targeted advertising
- Cross-device tracking

For neurodivergent families, this creates:
1. **Sensory intrusion**: High-stimulus targeted ads
2. **Anxiety**: Panopticon effect ("being watched")
3. **Energy drain**: Constant digital "noise"

### 3.2 The Solution: The HDMI EDID Emulator

The Phenix Phantom hardware interposes between the content source and the TV:

```
┌──────────────┐     ┌─────────────────┐     ┌─────────────┐
│   Content    │     │  PHENIX PHANTOM │     │   Smart TV  │
│   Source     │ ──► │  EDID Emulator  │ ──► │             │
│  (Console,   │     │                 │     │   (ACR      │
│   Streaming) │     │  Spoofs TV ID   │     │   Blocked)  │
└──────────────┘     └─────────────────┘     └─────────────┘
```

**Mechanism**:
- Device appears as "generic monitor" to source
- TV receives video but cannot fingerprint content
- User experience unchanged, surveillance blocked
