# Attractor

Calibration system for harmonic resonance training.

## Overview

Attractor is the calibration system that trains harmonic resonance. It converges toward a target resonance frequency (≈0.35) using strange attractor dynamics.

## Features

- Harmonic resonance training
- Target frequency: ≈0.35
- Strange attractor dynamics
- Calibration automation
- Resonance monitoring

## Architecture

Attractor uses:
- Harmonic analysis
- Frequency domain processing
- Resonance detection
- Calibration algorithms

## Integration

Attractor integrates with:
- **The Centaur**: Backend processing
- **The Scope**: Visualization
- **Node One**: Hardware calibration
- **The Buffer**: Signal processing

## Usage

Attractor operates during system calibration. It:
1. Analyzes harmonic content
2. Detects resonance frequencies
3. Adjusts system parameters
4. Converges toward target frequency
