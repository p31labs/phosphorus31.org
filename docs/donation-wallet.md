# Donation Wallet

ERC-5564 stealth address wallet system with ESP32-S3 hardware root of trust.

## Overview

The Donation Wallet implements ERC-5564 stealth addresses with hardware root of trust on ESP32-S3. It provides sovereign donation receiving with privacy-first design.

## Features

- ERC-5564 stealth addresses
- ESP32-S3 hardware root of trust
- Manifest V3 browser extension
- Delta topology
- Privacy-first design

## Architecture

The wallet uses:
- **Stealth Addresses**: ERC-5564 standard
- **Hardware Root of Trust**: ESP32-S3
- **Browser Extension**: Manifest V3
- **Delta Topology**: Network structure

The wallet integrates with:

- **Node One**: Hardware device (optional)
- **Blockchain**: Ethereum network
