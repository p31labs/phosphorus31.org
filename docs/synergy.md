## Usage

### Get Synergy Boost

```typescript
const engine = new GameEngine();
await engine.init();

// Get synergy boost for a system
const boost = engine.getSynergyBoost('p31-language');
console.log(`P31 Language boost: ${boost}x`);
```

### Create Emergent Behavior

```typescript
const synergy = engine.getSynergyManager();
const emergent = synergy.createEmergentBehavior([
  'p31-language',
  'cosmic-transitions',
  'family-vibe-coding'
]);

console.log(emergent.name);
console.log(emergent.power);
console.log(emergent.description);
```

### Get Synergy Statistics

```typescript
const stats = synergy.getStats();
console.log(`Nodes: ${stats.nodes}`);
console.log(`Connections: ${stats.connections}`);
console.log(`Loops: ${stats.loops}`);
console.log(`Active Loops: ${stats.activeLoops}`);
console.log(`Synergy Level: ${(stats.synergyLevel * 100).toFixed(1)}%`);
```

## Synergy Level

Total synergy is calculated from:

- **Total Power** - Sum of all system powers
- **Total Connections** - Number of connections
- **Total Amplification** - Sum of amplification strengths
- **Active Loops** - Number of active feedback loops

```
synergy = totalPower * (1 + connections * 0.1) * (1 + amplification * 0.1)
synergyLevel = min(1.0, synergy / 1000)
```
