# PART III: ROLE DEFINITIONS

## BOARD OF DIRECTORS

The Board exists to satisfy IRS requirements and provide governance oversight. It is NOT an operational team. The Board meets quarterly (minimum) and votes on:
- Annual budget approval
- Officer appointments
- Bylaw amendments
- Conflicts of interest
- Major partnerships or grants > $10,000

### Requirements (IRS / Georgia Law)
- Minimum 3 directors for 501(c)(3) credibility (Georgia allows 1, IRS wants 3+)
- Majority must be UNRELATED to the CEO by blood or marriage
- Each director has fiduciary duty: duty of care, duty of loyalty, duty of obedience
- Directors serve without compensation (standard for small nonprofits)

## SOP-003: Grant Application

**Trigger:** Grant opportunity identified with deadline > 14 days out.

1. **SCOUT** (Day 1): Will or Gemini identifies opportunity. Log in `20_GRANTS/` spreadsheet: funder, deadline, amount, requirements, fit score (1-5).
2. **ASSESS** (Day 1-2): If fit score ≥ 3, Will reviews full RFP. Decides GO/NO-GO.
3. **OUTLINE** (Day 2-3): Will creates application outline using funder's required format. Maps P31 capabilities to requirements.
4. **DRAFT** (Day 3-7): Will + Claude/Gemini draft each section. Key sections:
   - Executive Summary (always written last)
   - Problem Statement (communication overload for neurodivergent adults)
   - Solution Description (The Buffer, on-device scoring, privacy-first)
   - Organizational Capacity (board, technical expertise, open-source model)
   - Budget Narrative (from implementation blueprint cost model)
   - Evaluation Plan (user testing metrics, accessibility scores)
5. **REVIEW** (Day 7-10): Board reviews budget section. Ashley reviews compliance language. Katen reviews accessibility/community language.
6. **POLISH** (Day 10-12): Final formatting, required attachments gathered:
   - [ ] IRS determination letter (or fiscal sponsor letter from HCB)
   - [ ] Board list with affiliations
   - [ ] Current budget
   - [ ] Organizational chart
   - [ ] Letters of support (Hunter/Tools for Life)
7. **SUBMIT** (Day 12-13): Submit via funder's portal. Log submission in spreadsheet.
8. **ARCHIVE** (Day 14): Full application package saved to `20_GRANTS/[funder_name]/`.

## SOP-008: Medical Emergency Protocol

**Trigger:** Calcium symptoms (tingling, cramping, muscle spasms, brain fog, confusion) OR meltdown/shutdown.

### Hypocalcemia Event
1. **IMMEDIATE:** Take calcium carbonate (Tums or supplement) + calcitriol if due.
2. **LOG:** Record in GENESIS_GATE Medical_Log: timestamp, symptoms, medications taken.
3. **NOTIFY:** Mama receives automated alert if severity ≥ 3 (scale 1-5).
4. **CANCEL:** All INTEGRATE phase work cancelled for remainder of day.
5. **MONITOR:** Recheck symptoms at 30 min, 1 hour, 2 hours.
6. **ESCALATE:** If symptoms worsen or don't improve in 1 hour → ER. No exceptions.

### Meltdown/Shutdown
1. **IMMEDIATE:** Stop all work. Close all screens. Remove stimulation.
2. **LOG:** GENESIS_GATE auto-sets spoons to 0 if triggered via dashboard.
3. **NOTIFY:** Mama and Ashley notified: "Operator in shutdown. No contact until ALL CLEAR."
4. **RECOVER:** Minimum 2 hours no-contact. Breathing exercises if able. Weighted blanket. Dark room.
5. **RESUME:** Only VACUUM phase permitted for remainder of day. No INTEGRATE.
6. **DEBRIEF:** Next day, log trigger if identifiable. Update HOSTILE_SENDERS or spoon costs if needed.
