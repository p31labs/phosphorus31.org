## THE PROBLEM

1 in 36 children are diagnosed autistic. Most assistive technology is designed BY neurotypical people FOR neurotypical assumptions about disability. Late-diagnosed adults fall through every crack — too "functional" for traditional services, too disabled for traditional employment. When systems fail (divorce, job loss, benefits disruption), neurodivergent people lose their ground reference. There's no technology layer to catch them.

## THE STACK

```
┌──────────────┬──────────────┬──────────────┐
│  NODE ONE    │ THE BUFFER   │  THE SCOPE   │
│  Hardware    │  Software    │  Dashboard   │
│              │              │              │
│  ESP32-S3    │  Voltage     │  1,888       │
│  LoRa Mesh   │  Scanner     │  lines       │
│  Haptics     │  ADA Docs    │  6 systems   │
│  Offline     │  Auto-Hold   │  Production  │
└──────────────┴──────────────┴──────────────┘
```

**Node One:** ESP32-S3 handheld device for sensory regulation and emergency communication. Works offline, no cloud dependency. Haptic feedback (DRV2605L) and LoRa mesh networking (Meshtastic). Single-purpose tool that works when verbal communication fails.

**The Buffer:** Communication triage system that scans incoming messages for coercion patterns (urgency, shame, threats). Assigns voltage scores (0-10) and holds high-voltage messages (≥6) until operator is ready. Automatically generates ADA accommodation documentation.

**The Scope:** Daily operating dashboard — 1,888 lines of production code (Google Apps Script). Tracks spoons (executive function capacity), medications with 4-hour calcium-stimulant gap enforcement, coherence monitoring, and system alerts. In daily production use since late 2025.
