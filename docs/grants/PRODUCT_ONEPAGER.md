## THE PROBLEM

1 in 36 children are diagnosed autistic. Most assistive technology is designed BY neurotypical people FOR neurotypical assumptions about disability. Late-diagnosed adults fall through every crack — too "functional" for traditional services, too disabled for traditional employment. When systems fail (divorce, job loss, benefits disruption), neurodivergent people lose their ground reference. There's no technology layer to catch them.

## THE STACK

```
┌──────────┬──────────┬──────────┐
│ NODE ONE │THE BUFFER│THE SCOPE │
│ Hardware │ Software │Dashboard │
│ ESP32-S3 │ Voltage  │ 1,888    │
│ LoRa     │ Scanner  │ lines    │
│ Haptics  │ ADA Docs │ 6 systems│
└──────────┴──────────┴──────────┘
```

**Node One:** ESP32-S3 handheld device for sensory regulation and emergency communication. Works offline, LoRa mesh networking, haptic feedback. Purpose-built tool, not a phone app.

**The Buffer:** Communication triage system that scans incoming messages for coercion patterns (urgency, shame, threats). Assigns voltage scores (0-10), holds high-voltage content, generates ADA accommodation documentation automatically.

**The Scope:** Daily operating dashboard — 1,888 lines of production code. Tracks spoons (executive function capacity), medications with 4-hour calcium-stimulant gap enforcement, coherence monitoring, system alerts. In daily production use.

## ARCHITECTURE PRINCIPLES

**Delta Topology:** Mesh networking, no single point of failure. Everything works offline. No cloud dependency. When systems fail, P31 provides the technology layer to catch neurodivergent people.

**Mission Lock:** Nonprofit structure (501(c)(3)) ensures tools remain accessible. Open source + defensive publication prevents patent trolling. If P31 fails, the community can fork.

**Lived Experience:** Built BY an autistic person FOR autistic people. Every design decision accounts for sensory sensitivity, executive function challenges, and communication differences.
