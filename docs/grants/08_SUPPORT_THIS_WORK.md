# Support This Work

P31 Labs is pre-revenue, built by one person, and powered entirely by the belief that neurodivergent adults deserve tools designed for how their brains actually work. Every dollar funds what matters: hardware prototypes, server costs, and the time it takes to build technology that should already exist. All donations are tax-deductible through our fiscal sponsor, The Hack Foundation (EIN: 81-2908499).
