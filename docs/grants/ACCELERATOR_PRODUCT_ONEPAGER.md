## Key Points for One-Pager

### Problem
- Assistive technology market ignores intersection of autism, ADHD, and chronic medical conditions
- Existing tools designed for neurotypical users
- Late-diagnosed adults fall through every crack
- No technology layer when systems fail

### Solution
- **Node One:** Offline sensory regulation device (ESP32-S3, LoRa mesh)
- **The Buffer:** Communication voltage scanner with AI-powered coercion detection
- **The Scope:** Production dashboard (1,888 lines) managing medications, energy, executive function

### Founder Credibility
- 16 years DoD civilian submarine electrician (safety-critical hardware)
- Diagnosed AuDHD at 39 after unaccommodated federal service
- Built working MVP while on SNAP benefits
- Technical depth + lived experience = founder-market fit

### Traction
- Production code in daily use
- Working hardware prototype
- Defensive publications established
- Ecosystem partnerships formed

### Business Model
- Nonprofit (501(c)(3)) ensures mission lock
- Revenue: Hardware sales, SaaS licensing, grants, donations
- Open source + defensive publication prevents patent trolling
- Community-owned IP

- Go-to-market strategy
- Manufacturing partnerships
- Regulatory mentorship

- Network access
