The "Type B" variant exposes a specific 32-pin GPIO header that allows for the separation of these buses, enabling the simultaneous operation of the high-refresh UI and the high-speed radio.
         * Processor: Dual-core Xtensa LX7 running at 240 MHz, providing sufficient SIMD vector extension capability to perform the real-time matrix operations required for the Fisher-Escolà probability calculations.

         * Interface: SPI (Serial Peripheral Interface).
