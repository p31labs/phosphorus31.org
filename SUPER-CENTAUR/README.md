# The Centaur

Backend AI protocol system for P31. Human + Synthetic = Centaur.

## Overview

The Centaur is not a pair, but a protocol. It represents the interface between human operators and synthetic intelligence. The protocol is vendor-agnostic - cloud when connected, local when sovereign, hybrid when in between.

## Features

- Vendor-agnostic AI integration
- Cloud/Local/Hybrid operation
- Quantum brain integration
- Decision engine (MCDA)
- Knowledge graph (Neo4j)
- SOP generation
- Consciousness monitoring
- Legal & medical document processing
- Blockchain integration
- Family support systems

## Quick Start

See [setup.md](setup.md) for complete setup instructions.

### Prerequisites

- Node.js 18.0.0+
- PostgreSQL or SQLite
- Redis
- Neo4j (optional, for knowledge graph)

### Installation

```bash
npm install
cp .env.example .env
# Edit .env with your configuration
npm run db:init
npm run dev
```

## Architecture

The Centaur consists of:

- **Quantum Brain**: Decision engine, consciousness monitor, SOP generator
- **Legal Services**: Document processing, legal AI
- **Medical Services**: Medical documentation, health tracking
- **Blockchain Services**: Smart contracts, wallet integration
- **Family Support**: Family coordination, communication
- **Security**: Authentication, encryption, key management
