**Tone:** Real, working prototype. No polish. Show the actual system.

## PRODUCTION NOTES

### Camera Work
- Handheld, slightly shaky is fine — shows it's real
- Close-ups of device interactions
- Screen recordings for Buffer/Scope interfaces
- Natural lighting in grocery store (fluorescent is fine)

### Audio
- Use Will's actual voice — no professional VO needed
- Keep ambient sounds (grocery store noise)
- Don't over-produce — this is a demo, not an ad

### Graphics/Text Overlays
- Simple, readable fonts
- P31 brand colors (Phosphorus Green #2ecc71, Calcium Blue #60a5fa)
- Don't animate text unnecessarily
- Show actual system data, not mockups

### Device Props
- Use actual Node One prototype (even if it's breadboard/perfboard)
- Show real Buffer interface (even if it's terminal/CLI)
- Show real Scope dashboard (even if it's development version)

### Editing
- Quick cuts are fine
- Don't over-edit — let the system speak for itself
- Keep transitions simple
- Total runtime: 2 minutes ± 10 seconds

### What NOT to Do
- ❌ Don't use stock footage
- ❌ Don't add dramatic music
- ❌ Don't polish it to look like a commercial
- ❌ Don't hide the fact it's a prototype
- ❌ Don't script Will's reactions — let him be real
