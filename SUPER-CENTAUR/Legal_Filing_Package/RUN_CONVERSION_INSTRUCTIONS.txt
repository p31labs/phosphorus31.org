### 6. TROUBLESHOOTING

**If the script doesn't run:**
- Right-click `convert_to_pdfs.bat` and select "Run as administrator"
- Ensure you have write permissions to the folder

**If PDFs don't generate:**
- Check that Pandoc is properly installed
- Verify all .md files exist in the same folder
- Look for error messages in the command window

**If formatting looks wrong:**
- The script uses standard court formatting (Georgia 12pt, 1.5 spacing)
- All PDFs are optimized for court filing requirements
