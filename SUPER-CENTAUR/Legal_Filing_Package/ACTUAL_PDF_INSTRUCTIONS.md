Since automated tools are not available, you can easily convert these HTML files to proper PDFs using your web browser. This method produces high-quality, properly formatted PDFs.

2. **Convert each file to PDF**
   - Double-click any file to open it in your default browser
   - Press `Ctrl+P` (or `Cmd+P` on Mac) to open the print dialog
   - In the "Destination" or "Printer" section, select:
     - **Windows:** "Microsoft Print to PDF"
     - **Mac:** "Save as PDF"
     - **Chrome:** "Save as PDF" (under "Local Destinations")

   - Choose a location to save the PDF
   - Give it the same name as the original file
