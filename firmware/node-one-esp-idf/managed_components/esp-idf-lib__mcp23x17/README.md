Driver for I2C/SPI 16 bit GPIO expanders MCP23017/MCP23S17.

* [Documentation](https://esp-idf-lib.github.io/mcp23x17/)
* [Repository](https://github.com/esp-idf-lib/mcp23x17)
* [Issues](https://github.com/esp-idf-lib/mcp23x17/issues)

* [Component page at the ESP Component Registry](https://components.espressif.com/components/esp-idf-lib/mcp23x17)

## Installation

```sh
idf.py add-dependency esp-idf-lib/mcp23x17
```
