┌─────────────────────────────────────────────────────────────┐
│ ASSEMBLY SEQUENCE                                            │
├─────────────────────────────────────────────────────────────┤
│ 1. SMD Passives   → Resistors, capacitors (0805)           │
│ 2. SMD ICs        → TP4056, DRV2605L, regulator            │
│ 3. Modules        → ESP32-S3, LoRa module                   │
│ 4. LEDs           → SK6812 (heat sensitive, be careful!)    │
│ 5. USB-C          → Check for shorts!                       │
│ 6. Hotswap Sockets→ Kailh Choc sockets                      │
│ 7. Encoder        → EC11 rotary encoder                     │
│ 8. Headers        → JST battery, OLED pins                  │
│ 9. Switches       → Plug into hotswap sockets (no solder!)  │
│ 10. Test          → Power on, check voltages, program       │
└─────────────────────────────────────────────────────────────┘
