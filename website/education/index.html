Glossary

P31 NodeZero

The hardware device that acts as a mesh network node, featuring ESP32‑S3, LoRa, and haptics.

P31 Buffer

Software layer that processes and buffers signals in real time.

P31 Spectrum

User interface for visualizing mesh data and wallet information.

P31 Compass

The mesh routing protocol based on a modified AODV with topology awareness.

LOVE Token

Economic token of the P31 ecosystem, earned through Proof of Care.

Proof of Care

A metric that rewards nodes for contributing to mesh health (uptime, relays, etc.).

Frequently Asked Questions

Do I need a P31 NodeZero to participate?

You can explore the documentation and wallet without hardware. To join the mesh network, a P31 NodeZero is required.

Is the technology open source?

Yes, all code and hardware designs are licensed under Apache 2.0. See our Legal page.

How do I earn LOVE tokens?

By running a P31 NodeZero and participating in the mesh (relaying messages, maintaining uptime).

What accessibility features are included?

Screen reader support, high contrast mode, switch control, and haptic feedback. See our Accessibility statement.
