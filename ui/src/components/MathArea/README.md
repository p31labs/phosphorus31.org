# Math Area 🔢

A beautiful, accessible math learning and playing environment for young learners.

## Features

- **Calculator**: Full-featured calculator with basic operations
- **Number Line**: Interactive number line to explore positive and negative numbers
- **Shapes**: Visual shape learning with canvas drawing
- **Counting**: Interactive counting practice with visual feedback
- **Math Problems**: Practice problems with multiple choice or text input
- **Games**: Fun math games and activities
- **Score Tracking**: Track correct answers, total problems, and streaks
- **Accessible**: Full keyboard and screen reader support
- **Responsive**: Works on desktop, tablet, and mobile

## Modes

### 🧮 Calculator
- Basic arithmetic operations (+, -, ×, ÷)
- Clear, percentage, and sign change functions
- Large, easy-to-read display
- Keyboard support

### 📏 Number Line
- Interactive number line from -20 to 20
- Visual representation of numbers
- Input controls to set specific values
- Highlights current value and zero

### 🔺 Shapes
- Visual shape learning
- Circle, Square, Triangle, Rectangle, Star
- Canvas-based drawing
- Interactive selection

### 🔢 Counting
- Visual counting practice
- Add/subtract buttons
- Star visualization for each count
- Reset functionality

### ✏️ Problems
- Math problem generator
- Addition, Subtraction, Multiplication, Division
- Multiple choice or text input
- Instant feedback
- Score tracking with streaks

### 🎮 Games
- Quick access to all math activities
- Game cards for easy navigation
- Fun, engaging interface

## Usage

1. Click the "🔢 Math Area" button in The Scope
2. Select a mode from the mode selector
3. Start learning and playing!

## Keyboard Shortcuts

- Calculator: Use number keys and operators
- Problems: Enter key to submit answer
- Number Line: Arrow keys to navigate (coming soon)

## Score Tracking

The Problems mode tracks:
- **Correct**: Number of correct answers
- **Total**: Total problems attempted
- **Streak**: Current correct answer streak

## Accessibility

- Full keyboard navigation support
- Screen reader compatible
- High contrast mode support
- Reduced motion support
- Large, readable fonts
- Color-blind friendly design

## Technical Details

- Built with React and TypeScript
- Canvas API for shape drawing
- Local state management
- Responsive CSS with mobile support
- Smooth animations and transitions
