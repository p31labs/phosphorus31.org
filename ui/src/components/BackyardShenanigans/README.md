# Bash's Backyard Shenanigans 🌳

Interactive backyard play area built with love and light. As above, so below. 💜

## Overview

Bash's Backyard Shenanigans is an interactive, playful area where children can explore, play, and discover. It's designed to be fun, engaging, and educational.

## Features

### 🗺️ Explore
- Overview of all backyard activities
- Quick navigation to different areas
- Beautiful card-based interface

### 🌱 Garden
- Plant flowers and vegetables
- Build your own garden
- Track garden items
- Score points for planting

### 🐛 Bugs
- Interactive bug catching game
- Multiple bug types (ladybug, butterfly, bee, dragonfly, caterpillar)
- Animated bugs that move around
- Click to catch bugs and earn points
- Bugs regenerate after being caught

### 🌤️ Weather
- Change the weather (sunny, rainy, cloudy, windy)
- Visual weather effects
- Weather affects the background
- Educational weather information

### 🎮 Games
- Tag
- Target Practice
- Scavenger Hunt
- Obstacle Course

### ⚔️ Adventure
- Interactive adventure stories
- Multiple adventure paths
- Progress tracking
- Choose-your-own-adventure style

## Scoring System

- Plant garden items: +10 points
- Catch bugs: +5 points
- Complete activities: Various points

## Accessibility

- Full keyboard navigation
- Screen reader support
- High contrast mode
- Reduced motion support
- Touch-friendly interface

## Usage

1. Click "🌳 Bash's Backyard" button in The Scope
2. Choose an activity from the selector
3. Explore, play, and have fun!
