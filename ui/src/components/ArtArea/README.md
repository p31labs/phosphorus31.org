# Art Area 🎨

A beautiful, accessible drawing and painting canvas for young artists.

## Features

- **Drawing Canvas**: Smooth, responsive drawing with mouse and touch support
- **Color Palette**: 20 beautiful colors to choose from
- **Brush Sizes**: 7 different brush sizes (2px to 40px)
- **Save & Load**: Save artwork with custom names
- **Gallery**: View all saved artwork in a beautiful gallery
- **Download**: Download artwork as PNG files
- **Accessible**: Full keyboard and screen reader support
- **Responsive**: Works on desktop, tablet, and mobile

## Usage

1. Click the "🎨 Art Area" button in The Scope
2. Select a color from the palette
3. Choose a brush size
4. Start drawing on the canvas!
5. Save your artwork with a custom name
6. View your gallery to see all saved pieces

## Controls

- **Colors**: Click any color button to select it
- **Brush Size**: Click a brush size button to change brush size
- **Clear**: Clears the entire canvas
- **Save**: Saves the current artwork to the gallery
- **Gallery**: Opens the gallery to view all saved artwork
- **Download**: Downloads the current artwork as a PNG file

## Gallery

- Click "Gallery" to view all saved artwork
- Click on any artwork thumbnail to load it for editing
- Use "Edit" to load an artwork
- Use "Delete" to remove an artwork

## Accessibility

- Full keyboard navigation support
- Screen reader compatible
- High contrast mode support
- Reduced motion support
- Touch-friendly controls

## Technical Details

- Built with React and TypeScript
- Uses HTML5 Canvas for drawing
- LocalStorage for saving artwork
- Responsive CSS with mobile support
- Smooth drawing with line smoothing
