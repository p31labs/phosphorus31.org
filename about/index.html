About P31 Labs

Building assistive technology for neurodivergent minds. Open source. Offline-first. The mesh holds.

Mission

What We Do

P31 Labs builds open-source assistive technology for neurodivergent individuals. We create communication processing tools, sensory regulation devices, and executive function support systems that work offline, respect privacy, and form resilient mesh networks.

The problem: 15–20% of the global population is neurodivergent, yet there are zero open-source cognitive prosthetics. Commercial solutions are wellness apps with undefined FDA status, subscription fees, and cloud dependencies. They require smartphones, internet connections, and institutional permission — exactly what vulnerable populations often lack.

Our solution: Build the tools that don't exist yet. Communication prosthetics that process emotional subtext. Executive function scaffolds that work offline. Sensory regulation devices that don't require a smartphone. All open source. All mesh-resilient. All built by the people who need them most.

Local-First

Every system functions without cloud dependencies. Your data lives on your devices first. The cloud is for sync only, when you choose to connect. No vendor lock-in. No institutional dependency. No single point of failure.

Example: The Scope runs entirely in Google Apps Script (1,888 lines, production). Node One communicates via LoRa mesh — no internet required. The Buffer processes locally before any network transmission.

Privacy-First

Type-level encryption. Zero-knowledge verification. No telemetry. No analytics. No surveillance. The mesh protects its own. We don't collect data because we don't need to. We don't track users because we don't want to.

Example: The Buffer uses voltage assessment algorithms to detect coercive patterns without storing message content. L.O.V.E. Economy uses Proof of Care (proximity + HRV sync) without location tracking.

Mesh-Resilient

Tetrahedron topology. No single point of failure. Every node is a peer. When the neutral fails, the mesh holds. Inspired by R. Buckminster Fuller's geodesic geometry — the same math that makes domes strong makes networks resilient.

Example: Node One devices form LoRa mesh networks. If one node fails, messages route through others. The tetrahedron (4 vertices, 6 edges) is the minimum stable volume — the atomic unit of resilience.

Values

Core Principles

Open Source by Default

All software: Apache 2.0. All hardware: CERN-OHL-S. All research: CC-BY-4.0. Defensive publication to Zenodo with timestamped DOIs. We publish to prevent corporate patenting, not to profit from IP.

Built By Need

The founder is AuDHD, diagnosed at 39 after 16 years building workarounds that turned out to be assistive technology prototypes. Lived experience drives every design decision. Nothing about us without us.

Offline-First

Every system must function without internet. No cloud dependencies. No institutional permission required. When the grid fails, when the neutral is severed, when the system collapses — the mesh holds.

Universal Accessibility

WCAG 2.1 AA compliance. Age-adaptive (6 to 70+). Neurodivergent-first design. High contrast. Keyboard navigation. Screen reader support. Reduced motion options. The mesh holds for everyone.

Regulatory Transparency

Node One designed for FDA 510(k) pathway (21 CFR § 890.3710). All regulatory strategy documented. All clinical validation plans published. No hidden pathways. No regulatory shortcuts.

Nonprofit Structure

Georgia 501(c)(3) in formation. Fiscal sponsorship through Hack Club HCB. No equity. No exit strategy. No profit motive. The mission is the product. The mesh is the business model.
