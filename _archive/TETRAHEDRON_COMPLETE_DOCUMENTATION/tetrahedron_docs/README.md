## 🆘 WHEN THINGS GO WRONG

**First, don't panic.** Everything is fixable.

**Step 1:** Identify the problem
- Power issue?
- Screen issue?
- Radio issue?
- Software issue?

**Step 2:** Open the appropriate troubleshooting document
- **COMPLETE_TROUBLESHOOTING.md** has solutions for every common problem
- Follow the diagnostic flowcharts
- Test systematically

**Step 3:** Use the debugging mindset
- What was working before?
- What changed?
- Can we isolate the problem?
- What's the simplest test we can do?

**Step 4:** Model good problem-solving
- Stay calm
- Talk through your reasoning
- Celebrate when you figure it out
- Show that debugging is normal
