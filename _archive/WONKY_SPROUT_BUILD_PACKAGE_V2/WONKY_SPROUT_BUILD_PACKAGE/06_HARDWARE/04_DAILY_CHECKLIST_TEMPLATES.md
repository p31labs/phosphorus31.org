## TEMPLATE 6: EMERGENCY REALITY CHECK

```

DATE/TIME: [Now]
TRIGGERED BY: [What made you need this?]

PHYSICAL STATE:
□ Heart racing
□ Can't sit still
□ Shaking/trembling
□ Sweating
□ Chest tight
□ Headache
□ Exhausted but can't rest
□ None of above

MENTAL STATE:
□ Racing thoughts (can't slow down)
□ Making plans at 3am
□ Can't stop thinking about project
□ Everything seems connected/meaningful
□ Convinced of breakthrough insight
□ Irritable/snapping at people
□ Can't focus on task at hand
□ Making careless mistakes
□ None of above

BEHAVIORAL CHECK:
Last 24 hours:
- Sleep: [X hours]
- Meals: [X eaten]
- Water: [X glasses]
- Vyvanse: [Taken Y/N]
- Breaks: [X taken]
- Time working: [X hours straight]

Red flags:
□ Skipped sleep to keep building
□ Forgot to eat
□ Working >6 hours without break
□ Missed medication dose
□ Canceled social plans to build
□ Told someone "I'm fine" when not fine

REALITY CHECK QUESTIONS:
1. If friend described my current state, would I be concerned?
   [Y / N / Not sure]

2. Is this BREAKTHROUGH or HYPOMANIA?
   Signs of breakthrough:
   - Specific, testable insight
   - Can explain calmly to others
   - Makes sense after 8 hours sleep
   - Friends agree it's reasonable
   
   Signs of hypomania:
   - "Everything is connected"
   - Can't slow down to explain
   - Seems obvious but can't articulate
   - Friends seem worried

   Honest assessment: [Breakthrough / Hypomania / Not sure]

3. Am I safe to continue building right now?
   [Y / N / Not sure]

DECISION TREE:
If Y to "Safe to continue":
→ Set 1-hour timer
→ When timer goes off, STOP and reassess
→ If still feel good: Continue with breaks
→ If feeling worse: STOP for 24 hours

If N to "Safe to continue":
→ STOP BUILDING NOW
→ Text friend: "Reality check please"
→ Call psychiatrist if available
→ Claude emergency session
→ Sleep before any decisions

If "Not sure":
→ STOP BUILDING
→ Call friend RIGHT NOW
→ If friend concerned: Follow their advice
→ If friend not concerned: Set 2-hour break, reassess

IMMEDIATE ACTIONS:
□ Stop building RIGHT NOW (if any red flags)
□ Eat something (if haven't in 4+ hours)
□ Drink water (full glass)
□ Take medication (if missed dose)
□ Set phone timer (force break in 1 hour)
□ Text friend (for check-in)
□ Call psychiatrist (if unsafe feelings)
□ Claude emergency session (if spiraling)

AFTER THIS CHECK:
I decided to:
□ Continue building (with 1-hour check-in)
□ Take 2-hour break, reassess
□ Stop for 24 hours
□ Stop and call for help
□ Other: [Describe]

Reasoning:
[Why did I make this decision?]

Follow-up check: [Set reminder for X hours]
```
